## What is Protected?

The patent application covers:

### Core Innovation
- **Chronometric Interferometry**: The fundamental method of using intentional frequency offsets to achieve sub-nanosecond synchronization
- **Beat Signal Processing**: Extraction of timing information from intentionally generated beat frequencies
- **Phase Evolution Analysis**: The mathematical framework for determining propagation delay from beat phase

### System Architecture
- **Two-Way Protocol**: The bidirectional measurement protocol for clock bias cancellation
- **Multi-Carrier Unwrapping**: Resolution of phase ambiguities using multiple frequency offsets
- **Distributed Consensus**: Variance-weighted network synchronization without master clocks

### Implementation Methods
- **Frequency Offset Selection**: Optimal selection algorithms for beat frequency generation
- **Phase Extraction**: Closed-form estimators for joint time-frequency determination
- **Hardware Compensation**: Methods for handling real-world impairments
