# Contributing

Fork, hack, pull request.

## Setup

```bash
git clone https://github.com/YOUR_USERNAME/driftlock-choir
cd driftlock-choir
pip install -r requirements.txt
PYTHONPATH=. pytest
```

## Guidelines

- Run tests before submitting
- Follow existing code style
- Add tests for new features
- Keep it simple
