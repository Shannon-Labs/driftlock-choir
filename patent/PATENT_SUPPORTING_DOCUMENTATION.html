Network Consensus Algorithm

Variance-Weighted Consensus Update:
x_i(k+1) = x_i(k) + ε Σ_j∈N_i W_ij (d_ij - (x_i(k) - x_j(k)))

Where:
• x_i = [ΔT_i, Δf_i] (clock offset, frequency error)
• W_ij = diag(1/σ²_τ, 1/σ²_Δf) (inverse variance weights)
• d_ij = [τ̂_ij - τ̂_ji, Δf̂_ij - Δf̂_ji] (pairwise measurements)
• ε = step size derived from network Laplacian eigenvalues

Beat Signal Phase Evolution

Beat Phase Equation:
φ_beat(t) = 2π Δf (t - τ_ij) + (θ_i - θ_j) + 2π f_i τ_ij

Closed-Form Estimator:
τ̂ = (φ_beat(T) - φ_beat(0) - 2π Δf T + (θ_j - θ_i))/(2π f_i)

Where:
• Δf = intentional frequency offset
• τ_ij = propagation delay
• θ_i, θ_j = initial phase offsets
• f_i = carrier frequency
• T = observation window
