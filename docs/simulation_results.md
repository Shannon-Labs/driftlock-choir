# Simulation Results

This document summarizes the simulation performance of the Driftlock Choir system.
All official performance metrics are maintained in the single source of truth:
[RESULTS_STATUS.md](RESULTS_STATUS.md).

The results presented there reflect the current baseline performance across various
tapped-delay-line (TDL) channel profiles. As of the latest run, all profiles are
flagged by performance guardrails, indicating that further refinement is needed,
particularly for multipath mitigation in challenging environments.
